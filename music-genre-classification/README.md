# 🎧 Music Genre Classification

A machine learning project that classifies music into genres using audio signal processing and feature extraction techniques.

---

## 🚀 Features

* 🎵 Extracts audio features (MFCC, Chroma, Mel Spectrogram)
* 🤖 Trains ML model (Random Forest)
* 📊 Evaluates performance (Accuracy, Classification Report)
* 🔮 Predicts genre of new audio files

---

## 🛠 Tech Stack

* Python
* Librosa (Audio Processing)
* Scikit-learn (ML Models)
* NumPy, Pandas

---

## 📂 Dataset

* GTZAN Dataset (10 genres)
* Each genre contains audio files (.wav)

---

## ⚙️ Installation

```bash
pip install -r requirements.txt
```

---

## ▶️ Run Training

```bash
python main.py
```

---

## 🔍 Predict Genre

```bash
python predict.py
```

---

## 📊 Output
