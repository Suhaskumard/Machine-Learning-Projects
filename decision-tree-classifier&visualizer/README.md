# Decision Tree Classifier & Visualizer

## 📌 Overview

This project demonstrates a simple Decision Tree Classifier using the Iris dataset. It trains the model, evaluates accuracy, and visualizes the decision tree.

## 📁 Folder Structure

```
project/
│
├── dtree.py
├── README.md
```

## ⚙️ Requirements

* Python 3.x
* pandas
* matplotlib
* scikit-learn

Install dependencies:

```
pip install pandas matplotlib scikit-learn
```

## ▶️ How to Run

1. Save the code as `dtree.py`
2. Run the program:

```
python dtree.py
```

## 📊 Output

* Accuracy of the model
* Decision tree visualization
* Sample prediction result

## 📂 Dataset

* Uses built-in Iris dataset (no external file required)

## 💡 Applications

* Classification problems
* Basic machine learning learning
* Academic projects
