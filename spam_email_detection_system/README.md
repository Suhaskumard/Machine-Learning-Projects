# 📧 Spam Email Detection System

A simple Machine Learning-based Spam Email Detector built using Python and Scikit-learn.

---

## 🚀 Features

- Detects whether an email is Spam or Not Spam
- Uses TF-IDF + Naive Bayes algorithm
- Text preprocessing for better accuracy
- Command Line Interface (CLI)
- Model evaluation with metrics
